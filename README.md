# 🛒 Ecommerce Sales Analysis

## 📌 Overview
An end-to-end ecommerce analytics project analyzing 1,200 orders across 2.5 years (January 2023 – June 2025) using Advanced Excel and SQL.This project transforms raw transactional data into actionable business intelligence through data cleaning, advanced analysis and interactive dashboard design.

---

## 📂 Dataset
| Field | Details |
|---|---|
| Period | January 2023 – June 2025 |
| Total Orders | 1,200 |
| Unique Customers | 1,189 |
| Columns | OrderID, Date, CustomerID, Product, Quantity, UnitPrice, TotalPrice, OrderStatus, PaymentMethod, ReferralSource, CouponCode |

---

## 🛠️ Tools Used
| Tool | Purpose |
|---|---|
| Advanced Excel | Dashboard, Pivot Tables, RFM Segment, Cohort Analysis |
| SQL | Data Exploration, Aggregations, Customer Analysis |

---

## 📊 What I Built

### Advanced Excel
- **5 Pivot Table Analyses** — Monthly Revenue, Product Performance, 
  Order Status, Referral Source, Coupon Impact
- **RFM Customer Segmentation** — Scored 1,189 customers across 
  Recency, Frequency and Monetary dimensions
- **30 Month Cohort Retention Analysis** — Heatmap revealing 
  near-zero customer retention
- **Interactive KPI Dashboard** — 8 dynamic KPI cards, 4 charts, 
  2 slicers connected across all pivots

### SQL
- Data exploration and validation
- Revenue aggregations by product, channel and status
- Customer behaviour analysis
- Null value handling and duplicate detection

---

## 💡 Key Findings
- 📉 **41.4%** of orders cancelled or returned — major revenue leakage
- 🔄 **Near zero retention** — 0.93% customers return after first purchase
- 📱 **Instagram** drives highest revenue at ₹2,75,285
- 💰 **₹2,56,328** pipeline revenue waiting to be converted
- 🪑 **Chair** is top revenue product at ₹1,95,620
- 🏷️ **FREESHIP** outperforms SAVE10 — customers prefer free shipping
